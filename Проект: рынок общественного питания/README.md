## Изучение рынка заведений общественного питания в Москве

**Цель проекта** - провести комплексный исследовательский анализ рынка заведений общественного питания в Москве на основе данных Яндекс Карт и Яндекс Бизнеса (лето 2022 года), чтобы предоставить инвесторам фонда Shut Up and Take My Money обоснованные рекомендации по открытию нового заведения.

**Задачи проекта**:
* Изучить структуру и особенности рынка общепита Москвы: распределение заведений по категориям, районам, сетевому статусу и другим ключевым параметрам.
* Выявить наиболее перспективные сегменты рынка (категории заведений, районы города) с точки зрения спроса, конкуренции и ценовой политики.
* Определить факторы, влияющие на рейтинги заведений (корреляции с категорией, расположением, количеством мест и т. д.).
* Сформировать практические рекомендации для инвесторов по выбору типа заведения, локации, ценовой категории и других стратегических параметров.

**Основные этапы проекта**:
1. Загрузка данных и первичное знакомство с ними
2. Предобработка данных, включая проверку корректности данных, их типов, наличия или отсуствия пропусков
3. Исследовательский анализ данных
4. Итоговый вывод и рекомендации

**Инструменты**: Python (Pandas, matplotlib, seaborn). 

Проект реализован в рамках курса "Аналитик данных" (Яндекс.Практикум) в jupyter notebook.
