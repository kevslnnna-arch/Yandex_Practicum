## Исследование развития игровой индустрии в 2000-2013 гг.

**Цель проекта** - исследовать развитие игровой индустрии в 2000-2013 гг. (платформы, продажи по жанрам, предпочтения и т.д.)

**Основные этапы проекта**:
1. Загрузка данных и первичное знакомство с ними
2. Предобработка данных, включая проверку корректности данных, их типов, наличия или отсуствия пропусков
3. Фильтрация данных по временному периоду
4. Категоризация данных

**Инструменты**: Python (Pandas, numpy)
