## Проект: TED-конференции

**Ссылка на дашборд**: https://datalens.yandex/6nmyhru80hhyr?tab=2D

<img width="1891" height="944" alt="image" src="https://github.com/user-attachments/assets/bc738554-f421-4053-ae9d-4d4d85023622" />

**Цели проекта** - выявить ключевые показатели успешности выступлений (длительность, вовлечённость аудитории, тематика) и транслировать их в формат дашборда.

**Основные этапы проекта**:
1. Выявить ключевые метрики для организации конференции:
- популярные темы и реакция зрителей на них;
- среднее количество выступающих на конференции;
- средняя длина выступления;
- оптимальное распределение времени конференции.
2. Исследовать выступления, выделив:
- топ-10 самых смешных выступлений;
- топ-20 самых популярных тегов.
3. Проанализировать профили выступающих, выявив род деятельности тех, кому аплодируют чаще всего.
4. Обеспечить возможность детального просмотра информации о любом выступлении при обнаружении интересных инсайтов.

**Инструменты**: Yandex Datalens, SQL

Проект реализован в рамках курса "Аналитик данных" (Яндекс.Практикум).
