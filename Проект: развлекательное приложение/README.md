## Разработка A/B-тестирования и анализ результатов развлекательного приложения

**Цель проекта** - провести A/B‑тест нового алгоритма рекомендаций в развлекательном приложении и на основе статистически значимых данных определить, улучшает ли он ключевые продуктовые метрики по сравнению с текущей версией. По результатам теста сформулировать рекомендации для команды разработчиков о целесообразности внедрения нового алгоритма в продакшн.

**Основные этапы проекта**:
1. Подготовка и планирование A/B‑теста
   * Сформулировать гипотезу
   * Выбрать ключевые метрики для оценки
   * Определить параметры теста
   * Настроить разделение на группы
   * Настроить сбор данных
2. Проведение теста и мониторинг
3. Анализ результатов
4. Формулирование выводов и рекомендаций

**Инструменты**: Python (Pandas, matplotlib)

Проект реализован в рамках курса "Аналитик данных" (Яндекс.Практикум) в jupyter notebook.
