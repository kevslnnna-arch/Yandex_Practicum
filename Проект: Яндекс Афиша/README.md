## Исследовательский анализ сервиса Яндекс Афиша

**Ссылка на дашборд**: https://datalens.yandex/lre9oucoqlsu5

<img width="1883" height="937" alt="image" src="https://github.com/user-attachments/assets/fc67b0d2-2d1c-4c49-b0a3-baf8faa113a8" />

**Цель проекта** - комплексно проанализировать динамику продаж билетов на мероприятия в сервисе Яндекс Афиша осенью 2024 года, выявить ключевые тренды и факторы, влияющие на изменение пользовательских предпочтений, а также предоставить команде продукта инструменты для мониторинга и принятия решений в преддверии периода распродаж и новогодних акций..

**Конкретные цели**:
* Создать интерактивный дашборд в Yandex DataLens для отслеживания ключевых бизнес‑показателей сервиса Яндекс Афиша в режиме реального времени.
* Провести углублённый исследовательский анализ данных в Python, чтобы выявить закономерности в динамике продаж, популярности мероприятий и поведении пользователей.
* Проверить гипотезу о различии в поведении пользователей мобильных устройств и стационарных компьютеров.
* Сформулировать практические рекомендации для команды продукта по оптимизации стратегии продаж в предновогодний период.

  
**Основные этапы и задачи проекта**:
1. Анализ данных с помощью SQL и создание дашборда в Yandex DataLens:
   * Знакомство с данными
   * Расчёт ключевых метрик с помощью SQL
   * Подготовка данных для дашборда
   * Создание дашборда в Yandex DataLens
2. Исследовательский анализ данных и проверка гипотез с помощью Python:
   * Загрузка и предобработка данных
   * Исследовательский анализ
   * Проверка гипотез
   * Анализ сезонности и трендов
   * Формулирование выводов и рекомендаций
   * Оформление итогового отчёта

**Инструменты**: Python (Pandas, matplotlib, numpy, scipy.stats). 

Проект реализован в рамках курса "Аналитик данных" (Яндекс.Практикум) в jupyter notebook.
