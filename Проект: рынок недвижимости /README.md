## Анализ рынка недвижимости в Санкт-Петербурге и Ленинградской области

**Ссылка на дашборд**: https://datalens.ru/8swg6xm04en4s-dashbord-dlya-agentstva-nedvizhimosti

**Цель проекта** — провести комплексный анализ рынка жилой недвижимости Санкт‑Петербурга и Ленинградской области на основе архивных данных сервиса «Яндекс Недвижимость», чтобы помочь агентству недвижимости сформировать эффективную бизнес‑стратегию при выходе на новый рынок.

**Задачи**:
1. Выявить наиболее перспективные сегменты рынка жилой недвижимости (по типам объектов, районам, ценовым категориям и т.д.) для фокусировки бизнес‑усилий агентства.
2. Определить сезонные закономерности активности на рынке недвижимости (периоды пикового спроса и предложения), чтобы оптимально планировать маркетинговые кампании и сроки публикации объявлений.
3. Предоставить агентству наглядную аналитическую отчётность (дашборд) для мониторинга ключевых показателей рынка и принятия управленческих решений.

**Основные этапы проекта**:
1. Исследовательский анализ данных.
2. Решение ad hoc задач с помощью SQL.
3. Доработка дашборда в DataLens.

**Инструменты**: Yandex DataLens, SQL, Python, DBeaver
