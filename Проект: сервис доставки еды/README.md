## Анализ ключевых метрик сервиса доставки еды «Всё.из.кафе» в Саранске

**Ссылка на дашборд**: [https://datalens.yandex/6nmyhru80hhyr?tab=2D](https://datalens.yandex/a9ana4stdyw4u?tab=5m)

**Цели проекта** - разработать информативный дашборд «Ключевые бизнес‑метрики в Саранске (май — июнь 2021)» и составить аналитический отчёт на его основе, чтобы оценить текущее состояние клиентской базы сервиса доставки еды «Всё.из.кафе» и выявить ключевые тренды и аномалии за период с начала мая до конца июня 2021 года. Это позволит принимать обоснованные управленческие решения для повышения эффективности бизнеса.

**Основные этапы проекта**:
1. Подготовка SQL‑запросов для расчёта ключевых метрик
   * Рассчитать DAU (Daily Active Users)
   * Рассчитать Conversion Rate (коэффициент конверсии)
   * Рассчитать средний чек
   * Рассчитать LTV (Lifetime Value)
   * Рассчитать Retention Rate (коэффициент удержания)
2. Создание визуализаций в DataLens
3. Сборка дашборда
4. Анализ данных и подготовка аналитической записки

**Инструменты**: Yandex Datalens, SQL

Проект реализован в рамках курса "Аналитик данных" (Яндекс.Практикум).
